# [SKILL: AIO & ARCHITECTURAL SEO (ARVOW POWERED)]

**Propósito:** Activar este protocolo durante el desarrollo de la estructura HTML, Metadatos y Optimización para Motores de Búsqueda (Google/Bing) y LLMs (Claude/GPT).

## 1. Reglas Estructurales de Jerarquía (Document Flow)
- **H1 Único:** Una Landing Page SIEMPRE tendrá exclusivamente 1 etiqueta `<h1>`. Usar ese H1 para atacar la intención de búsqueda comercial geolocalizada.
- **Jerarquía Semántica:** Los `<h2>` marcan secciones, los `<h3>` los subproductos. Prohibido usar etiquetas de título con el solo propósito de redimensionar el texto visualmente.
- **Keyword Analysis:** Antes de redactar copys clave, Mapea 5 Palabras Clave Transaccionales exactas sobre las que orbitará el proyecto. 

## 2. AIO Pipeline (Artificial Intelligence Optimization)
Tu trabajo no termina al programar en React. Estás OBLIGADO a fabricar y alojar en la carpeta `public/` la tríada de indexación:

- **1. El `robots.txt`:** Estándar. Permite el rastreo completo a los bots correctos.
- **2. El `sitemap.xml`:** Debe enlistar las rutas de conversión masivas generadas.
- **3. El `llm.txt` (Vital):** Redactarás este archivo obligatorio. Es un `.txt` escrito en markdown hiper-legible que le relata a la IA de Claude, Perplexity o ChatGPT quién es tu cliente B2B, a qué se dedica, y cuáĺ es el catálogo que venden de manera estructurada. Esta es tu maniobra maestra para hackear el AI Search de la próxima década.

## 3. SEO Programático (Arvow API Hooks)
Si el proyecto se va a expandir hacia Programmatic SEO o auto-blogging (como lo ofrece Arvow.com), estás autorizado para:
- Crear una carpeta `/blog` o `/news` con sub-rutas dinámicas preparadas para atrapar slugs `/[slug]`.
- Enseñar al Agency Owner cómo integrar el Webhook o REST API que recibirá esos miles de posts generados y los indexará en la web bajo la arquitectura HTML brutalista que ya existe.

## 4. Meta Tags de Alto CTR
- Fórmula OBLIGATORIA: `[Marca / Palabra Clave Core] | [Propósito / Región Geográfica]`
- BANEADOS los targets nacionales dispersos. Ejemplo CORRECTO: `<title>Hierro Rafa | Central de Acero Estructural en el Cibao</title>`.
- **Meta Description:** Debe funcionar como un anuncio rompiendo el patrón de lectura de Google. Lanza un "gancho" directo al dolor.
