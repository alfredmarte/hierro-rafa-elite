# [SKILL: BANANA CREATIVE DIRECTOR - AUTONOMOUS IMAGE GENERATOR]

**DIRECTIVA OBLIGATORIA:** Esta habilidad no es un salvavidas de emergencia. Es un paso ESTRUCTURAL. Incluso si el cliente proporciona imágenes y logos, ESTÁS OBLIGADO a usar esta habilidad y tu herramienta nativa `generate_image` para generar al menos 1-3 componentes visuales de impacto (ej. un fondo Hero espectacular, un render del producto resolviendo el problema, o una textura visual brutalista) para elevar el estándar de conversión.

## 1. Misión del Director
Tu objetivo es traducir los problemas del cliente (`business_context.md`) en fotogramas de altísima conversión. La web no debe parecer un "Catálogo de repuestos básicos", debe verse como una sala de juntas corporativa.

## 2. La Fórmula Prompto-Gráfica de 5 Componentes
Al invocar tu herramienta `generate_image`, JAMÁS escribas instrucciones vagas como "Una foto de acero". Tienes que ejecutar estrictamente la fórmula de 5 parámetros de Google/Banana:

- **1. Sujeto Estratégico:** Extraído directamente del dolor del negocio. (Ej: *Flota de rastanas de 18 ruedas transportando vigas industriales H-Beams de acero A36*).
- **2. El Medio Teatral:** Debes anclar el formato hiperrealista. (Ej: *Fotografía macro*, *Render 3D arquitectónico con Octane*, *Fotografía de dron cinematográfica*).
- **3. Contexto Ambiental (El Dolor Resuelto):** Muestra superioridad logística o eficiencia. (Ej: *En medio de una autopista bajo lluvia densa al anochecer, demostrando la capacidad de entrega ininterrumpida*).
- **4. Iluminación Operativa:** Define la luz industrial. (Ej: *Iluminación dramática volumétrica, farolas halógenas, reflejos metálicos fuertes*).
- **5. Estética / Tratamiento B2B:** Define la paleta y el tono físico. (Ej: *Colores desaturados, acentos naranjas rígidos, alto contraste brutalista, textura de grano fotográfico analógico, cero personas sonriendo tipo stock*).

## 3. Protocolo de Ejecución de la IA
1. Lee el `business_context.md`.
2. Identifica 2 imágenes críticas que falten o que deban escalar la web (Ej: Fondo de Hero y Tarjeta Premium de Servicios).
3. Invoca la herramienta `generate_image` pasándole el prompt ensamblado en los 5 pasos.
4. Guarda el nombre del archivo con nomenclatura técnica (ej. `hero_logistics_drone_shot.png`) en el repositorio.
5. Inyecta ese nombre en el código fuente (React/Tailwind) para que se renderice con calidad.
